# 헬미 - 초간단 운동기록, 운동 타이머

복잡한 기능은 모두 뺀 초간단 운동 기록 메모장 어플입니다. 
Flutter로 제작하였으며 운동기록, 캘린더, 타이머 기능이 있습니다.
